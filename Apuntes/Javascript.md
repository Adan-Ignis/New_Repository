Es un lenguaje de programación con las siguientes caracteristicas:

    - Debilmente tipado
        No es necesario especificar el tipo de dato que es una variable, a comparación de C++ y JAVA (Que son fuertemente tipado) que necesita decir el tipo de dato que es cuando se declara la variable

    - Tipado dinamico
        Se puede cambiar el tipo de dato de una variable, a comparación de C++ y JAVA (Que son de tipado estatico) que no se puede cambiar

    - Sensible a mayusculas y minusculas

Formas de escribir los nombres de las variables, constantes, etc...

    - camel case
        function crearMensaje() {}

    - Pascal case
        function CrearMensaje() {}

    - Snake case
        function nombre_propio() {}

    - Kebab case
        function nombre-propio() {}
